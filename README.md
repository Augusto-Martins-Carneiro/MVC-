CRUD de Lanches

Projeto desenvolvido para a disciplina Desenvolvimento de Aplicação para Internet I, com o objetivo de colocar em prática um CRUD utilizando tecnologias básicas da web.

---

📌 Sobre o projeto

A ideia foi criar um sistema simples para gerenciar lanches, funcionando como um cardápio digital básico.

Com ele, o usuário pode cadastrar novos lanches, visualizar os que já existem, editar informações e excluir itens quando quiser.

Tudo isso funciona direto no navegador, sem precisar de servidor ou banco de dados externo.

---

🚀 Funcionalidades

- Adicionar novos lanches
- Visualizar a lista atualizada
- Editar informações de um lanche
- Remover lanches
- Informar os ingredientes de cada item

---

💻 Tecnologias utilizadas

- HTML para estruturar a página
- CSS para organizar e melhorar a aparência
- JavaScript para toda a lógica do sistema
- LocalStorage para armazenar os dados no navegador

---

🧠 Lógica do sistema

O projeto foi baseado no conceito de CRUD:

- Criar novos dados
- Ler (exibir) os dados na tela
- Atualizar informações
- Deletar registros

A lógica principal foi feita em JavaScript, que controla os dados e atualiza a interface conforme o usuário interage.

---

💾 Armazenamento

Os dados são salvos utilizando o LocalStorage do navegador, o que permite que as informações continuem disponíveis mesmo após atualizar a página.
